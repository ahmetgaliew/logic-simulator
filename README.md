# Симулятор логических вентилей

Простой учебный проект для моделирования **логических вентилей** (AND, OR, NOT, XOR) с графическим интерфейсом на **Python (Tkinter)**.

---

## Возможности

- Переключение входных сигналов (A и B)
- Выбор типа вентиля (AND / OR / NOT / XOR)
- Отображение результата в режиме реального времени

---

## Установка и запуск

1. Установите Python 3.11+  
2. Склонируйте репозиторий
3. Запустите main.py
